# Fastfood Rank

Учебный веб‑проект (Vite + React + TypeScript) с адаптивной версткой (320px+), CSS‑переменными по палитре и доступностью (ARIA).

## Страницы

- Главная: «Бренды» + «Таблица» (`#/`)
- Чемпионы: `#/champions`
- Акции и бонусы: `#/promos`
- Меню: `#/menu`
- Профиль: `#/profile`

## Скрипты

```bash
npm i
npm run dev
npm run build
npm run lint
```
