# Fastfood Rank

Учебный веб‑проект (Vite + React + TypeScript) с адаптивной версткой (320px+), CSS‑переменными по палитре, доступностью (ARIA) и unit‑тестами.

## Страницы

- Главная: «Бренды» + «Таблица» (`#/`)
- Чемпионы: `#/champions`
- Меню: `#/menu`
- Профиль: `#/profile`

## Скрипты

```bash
npm i
npm run dev
npm run build
npm test
npm run lint
```
